# WPoets Full Stack Developer Test

A PHP and MySQL based CRUD application developed as part of the WPoets Full Stack Developer assignment.

## Project Structure

* `index.php` – Frontend UI displaying tabs/accordion, content slider, and synchronized images.
* `admin.php` – Admin panel for managing slider content through CRUD operations.
* `api.php` – Backend API endpoints for data operations.
* `db.php` – Database connection and query handling.
* `config.php` – Application configuration.
* `style.css` – Custom styling.
* `schema.sql` – Database schema and sample data.
* `images/` – Project assets and uploaded images.
* `Answers to technical questions.md` – Responses to the assignment questions.

## Features

* Create, Read, Update, Delete (CRUD) operations
* MySQL database integration
* Responsive desktop and mobile layouts
* Tabs on desktop and accordion on mobile
* Synchronized content slider and image display
* Bootstrap and jQuery based implementation

## Setup

1. Clone the repository.
2. Import `schema.sql` into MySQL.
3. Update database credentials in `config.php`.
4. Start a PHP server.
5. Open:

   * `admin.php` for content management.
   * `index.php` for frontend preview.
